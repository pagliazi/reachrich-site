# FAQ

**Q: 数据从哪来？准不准？**
A: 多市场、多源数据平台，**多源交叉校验 + 除权复权一致 + 新鲜度监控 + 物理约束校验**。对外只暴露统一口径的数据契约，不暴露来源与实现。详见[稳定与准确](reliability.md)。

**Q: 实时和历史是一套数据吗？**
A: 是。同源同口径，复权一致，回测与盘中可直接衔接，避免"两套数据对不上"。

**Q: 系统稳定吗？**
A: 7×24 持续运行、自愈、多源容错，中央健康监测 + 告警。详见[稳定与准确](reliability.md)。

**Q: 怎么接入？**
A: API / MCP，统一数据契约；MCP 可让大模型直接、安全地取数。详见[接入方式](access.md)。具体端点/配额见 [reachrich.ai](https://reachrich.ai)。

**Q: 覆盖哪些市场和数据？**
A: A 股 / 港股 / 美股；行情、资讯、基本面、资金流、龙虎榜、概念板块、历史 K 线、除权复权等。详见[数据覆盖](data-coverage.md)。

**Q: 和 RR-Agent 是什么关系？**
A: ReachRich 是数据底座（被调用方）；RR-Agent 是上层量化研究智能体（调用方，独立项目）。职责分离、独立演进。

**Q: 会公开数据源 / 架构吗？**
A: 不会。对外只陈述能力与口径，不公开来源、主机、技术栈或部署细节。

**Q: 这是投资建议吗？**
A: 不是。本项目为**数据研究平台**，所有内容仅供数据研究用途，不构成投资建议、投资参考、荐股或收益承诺；盈亏由使用者自行判断与承担。
