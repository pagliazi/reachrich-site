# 接入方式

> 能力级说明。具体端点、鉴权、配额以 [reachrich.ai](https://reachrich.ai) 为准，本仓库不公开。

## API

- 以编程方式取数：行情、资讯、基本面、资金、板块等。
- **统一数据契约**：字段、单位、复权方式一致，调用方无需关心底层来源与差异。
- 实时与历史同口径，便于"回测—盘中"使用同一套取数逻辑。

## MCP（Model Context Protocol）

- 让大模型 / AI agent **直接、安全地调用数据能力**：把行情、新闻、因子原料作为 AI 的数据与决策来源。
- 适合构建"会看盘、会读新闻、会用因子"的智能体。

## 典型调用方

- 上层 **RR-Agent**（量化研究智能体，独立项目）通过 API / MCP 取数，完成因子挖掘、选股、组合优化与执行。
- 第三方研究 / AI agent 也可作为数据消费方接入。

```
数据消费方（RR-Agent / 研究脚本 / AI agent）
        │  API / MCP（统一数据契约）
        ▼
   ReachRich 数据平台
```

> 端点、鉴权与配额属产品配置，不在本展示仓库公开；不涉及主机、IP 或部署细节。
