📖 A 股量化数据术语词典

30 个 A 股量化数据工程相关术语,中英对照定义。聚焦数据底层、行情微结构、基本面与资金、接入工程。综合量化研究术语见 RR-Agent 完整词典(62 术语,跨域)。

📊 数据底层 / Data Layer

A 股 · China A-shares
沪深 A 股 + 创业板 / 科创板 / 北交所。
港股 · Hong Kong stocks
香港主板与创业板上市证券。
美股 · US stocks
NYSE / NASDAQ 等美国主要交易所证券。
除权除息 XDXR · XDXR → 详解
送股/转增/派息导致的非交易性价格跳空。
前复权 · Forward-adjusted → 详解
以当前价向前调整,适合看图与近期分析。
后复权 · Backward-adjusted → 详解
以上市首日向后累乘,适合长周期回测。
数据契约 · Data contract
字段、单位、复权方式在所有源所有时点的统一约定。
多源校验 · Multi-source validation → 详解
多源比对,偏离超阈值自动标记或切换。
新鲜度 · Freshness
数据时效阈值监控,过期即告警。
物理约束 · Physical constraint
最高价≥最低价、量≥0 等规则拦截脏数据。

📈 行情与微结构 / Quotes & Microstructure

实时快照 · Real-time snapshot
秒级行情盘口聚合。
K 线 · K-line
日/分钟/周/月 OHLCV 蜡烛图。
逐笔成交 · Tick data → 详解
每笔成交时间、价、量。
集合竞价 · Call auction
开盘前与收盘前的集中撮合。
涨跌停 · Daily price limit
A 股 ±10%/ST ±5%/创科 ±20%/北交 ±30%。
Lee-Ready 算法 · Lee-Ready → 详解
主动买卖判定:成交价 vs 中间价 + tick test 兜底。
大单/超大单 · Large/super-large order → 详解
按单笔大小阈值分类(典型:超大单≥10w 股)。

💰 基本面与资金 / Fundamentals & Capital

F10 · F10 → 详解
上市公司基本面综合(财报/估值/分红/股本)。
ROE · Return on Equity → 详解
净利润/平均归母权益(TTM/单季/年三种口径需统一)。
EPS · Earnings per Share
每股收益。
BVPS · Book Value per Share
每股净资产。
北向资金 · Northbound
沪深港通流入 A 股的外资。
龙虎榜 · Dragon List
交易所每日异动股营业部明细。
大宗交易 · Block trade
场外大额成交,可选定价但有锁定期。

🔌 接入 & 工程 / Access & Engineering

API · HTTP REST API → 详解
程序化数据接入,token 鉴权。
MCP · Model Context Protocol → 详解
LLM 直接调用数据工具链的开放标准。
限流配额 · Rate limit
按 token 与时间窗的调用频次限制,防滥用。
情感分析 · Sentiment → 详解
新闻打分(词典法 vs ML)。
概念板块 · Concept board
热点主题(AI、新能源等)分组。
板块轮动 · Sector rotation
行业/概念间相对强弱周期切换。
